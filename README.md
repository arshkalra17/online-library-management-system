# Online Library Management System

## 1. Project Overview
The Online Library Management System is a web-based application designed to simplify and digitize library operations for small schools and colleges. It replaces manual record-keeping with an easy-to-use interface for managing books, users, and borrowing activities.

The system allows librarians to add, update, delete, and track books, while students can search for books and view their availability in real time.

---

## 2. Problem It Solves
Many educational institutions still rely on manual methods to manage library records, which leads to:
- Loss of data and incorrect book availability
- Time-consuming book issue and return processes
- Lack of transparency for students
- Difficulty in tracking overdue books

Existing commercial library systems are often expensive and overly complex, making them unsuitable for small institutions and student projects.

---

## 3. Target Users (Personas)

### Persona 1: Librarian
- Age: 30–55  
- Role: Manages library operations  
- Needs:
  - Add, update, and delete book records
  - Track issued and returned books
  - Monitor overdue books
  - Maintain accurate availability status  
- Pain Points:
  - Manual paperwork
  - Time-consuming record updates
  - Difficulty tracking lost or overdue books  

---

### Persona 2: Student
- Age: 16–25  
- Role: Library user  
- Needs:
  - Search books by title, author, or category
  - View book availability
  - Know due dates for borrowed books  
- Pain Points:
  - Not knowing if a book is available
  - Delays in getting book information
  - Manual inquiry process  

---

## 4. Vision Statement
To create a simple, affordable, and user-friendly online library system that improves accuracy, accessibility, and efficiency in library management for small educational institutions.

---

## 5. Key Features / Goals
- Secure login for librarians and students  
- Add, update, delete, and search books  
- Issue and return book tracking  
- Real-time book availability status  
- Due-date tracking for borrowed books  
- Optional export of records (PDF/CSV)  
- Simple notification system for overdue books  

---

## 6. Success Metrics
- System runs without major errors  
- Librarians can manage book records easily  
- Students can search and view availability successfully  
- Accurate storage of issue and return records  
- Project completed within 3 months  
- Positive feedback from test users  

---

## 7. Assumptions & Constraints

### Assumptions
- Users have access to a web browser  
- Basic internet connectivity is available  
- Users will provide accurate data  
- Librarians and students are willing to use the system  

---

### Constraints
- Must be completed within 3 months  
- Use only free and open-source technologies
